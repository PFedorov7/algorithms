Алгоритм Дейкстры.

Алгоритм Дейкстры состоит из четырех шагов:
1. Найти узел с наименьшеи ̆стоимостью (тоесть узел, до которого можно добраться за минимальное время).
2. Обновить стоимости соседей этого узла (вскоре я объясню, что имеется в виду).
3. Повторять,пока это не будет сделано для всех узлов графа.
4. Вычислить итоговый путь.

Для вычисления кратчайшего пути в невзвешенном графе используется поиск в ширину. Кратчайшие пути во взвешенном графе вычисляются по алгоритму Дейкстры.
В ненаправленном графе каждое новое ребро добавляет еще один цикл. Ал­ горитм Дейкстры работает только с направленными ациклическими графами.
Алгоритм Дейкстры не может использоваться при наличии ребер, имеющих отри­ цательный вес.
Если вы хотите найти кратчайший путь в графе, содержащем ребра с отрицательным весом, для этого существует специальный алгоритм, называемый алгоритмом Беллмана-Форда.
